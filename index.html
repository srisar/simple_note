it is a very simple note taking app without any fancy features, duh!

the purpose is to keep it simple and distraction-free and when resized to a smaller size, have a good ui that maximizes the tying area.

you can install this as an app on windows desktop if you are using google chrome.

it uses jquery, bootstrap, lodash, popper.js

also utilises html5 local storage to save the notes locally.
